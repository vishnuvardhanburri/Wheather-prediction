Project details

WeatherML live Python AIML app

A complete live weather prediction system built with a Python inference API, multi-page dashboard, explainability, alerts, reports, Docker support, and mobile-ready PWA behavior.

Live demo

wheather-prediction.onrender.com

Developed by Vishnu Vardhan Burri. All rights reserved.

Why

Built as a working system

The app moves beyond a basic notebook by resolving real places, ingesting live weather data, scoring confidence in Python, and serving structured results to a polished browser experience.

Install

Windows, macOS, Android, Apple

WeatherML is configured as an installable PWA, so the live app can be added to desktop or mobile devices from supported browsers.

Run

Local, Docker, Render

Run locally with python3 server.py, in Docker with docker run -p 4173:4173 weatherml, or live on Render using HOST=0.0.0.0.
