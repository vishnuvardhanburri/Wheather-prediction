Why

Project story

WeatherML was built to move beyond a basic weather form or notebook. The goal was to create a live deployed app that combines Python APIs, future forecasts, model confidence, explainability, training artifacts, and deployment proof.

Read story

Reality check

Limitations

This app is not an official weather warning system. Forecast quality depends on external data, and the historical model registry is a transparent benchmark rather than a safety-critical model.

Read limitations

Testing

What was verified

The project was tested locally, in Docker, through GitHub Actions, and on Render. A real CI failure caused by external API timeout was fixed by moving CI to an offline fixture.
