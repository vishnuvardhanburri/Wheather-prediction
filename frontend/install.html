Cross-platform app

Install WeatherML on Windows, macOS, Android, and Apple devices.

WeatherML runs from the live Render deployment, but it can also be installed from the browser as a standalone PWA with app-style launching and cached shell assets.

Install availability depends on your browser and device policy.

Windows

Install from Edge or Chrome

Open the live demo, choose the browser install option from the address bar or menu, then pin WeatherML to Start or taskbar.

macOS

Add to Dock

Use Safari, Chrome, or Edge install support to save WeatherML as a standalone app and keep it beside your everyday tools.

Android

Add to home screen

Open WeatherML in Chrome, tap install or add to home screen, then launch it like a native weather app.

iPhone and iPad

Use Safari share

Open the live link in Safari, tap Share, choose Add to Home Screen, and launch WeatherML from the icon.
